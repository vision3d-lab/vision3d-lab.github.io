Abstract

Open-vocabulary 3D scene understanding enables users to segment novel objects in complex 3D environments through natural language. However, existing approaches remain slow, memory-intensive, and overly complex due to iterative optimization and dense per-Gaussian feature assignments.

To address this, we propose LightSplat, a fast and memory-efficient training-free framework that injects compact 2-byte semantic indices into 3D representations from multi-view images. By assigning semantic indices only to salient regions and managing them with a lightweight index-feature mapping, LightSplat eliminates costly feature optimization and storage overhead.

We further ensure semantic consistency and efficient inference via single-step clustering that links geometrically and semantically related masks in 3D. We evaluate our method on LERF-OVS, ScanNet, and DL3DV-OVS across complex indoor-outdoor scenes, achieving state-of-the-art performance with up to 50-400× speedup and 64× lower memory.

Why Existing Methods Do Not Scale

The bottleneck is not just accuracy. Existing open-vocabulary 3D pipelines usually pay for semantics in the most expensive possible way: repeated optimization, dense language features on every Gaussian, and supervision that is still tied too closely to 2D rendering.

Problem 1
Iterative feature optimization is too slow

Prior methods repeatedly render views and align them with CLIP embeddings. That loop becomes the dominant cost of feature distillation and makes practical deployment unnecessarily slow.

Problem 2
Per-Gaussian language features waste memory

Storing high-dimensional semantic features for every Gaussian creates heavy memory overhead and forces inference to compare text queries against a very large number of primitives.

Problem 3
2D supervision degrades 3D semantics

When semantics are learned indirectly through rendered 2D feature maps, overlapping Gaussians blur boundaries and weaken the geometric consistency that real 3D reasoning needs.

How LightSplat Works

LightSplat is designed around a direct response to the three bottlenecks above. Rather than optimizing dense semantic features on every Gaussian, it first injects only the mask identities that matter, then consolidates them into coherent 3D object clusters. This is the mechanism behind the paper's central claim: efficient 3D understanding without giving up object-level reasoning.

Figure 2. The pipeline is deliberately compact: extract SAM masks and CLIP features, inject only lightweight indices into 3D Gaussians, filter unreliable masks with 3D support, and build coherent object-level semantics through context-aware clustering.

1
Extract mask semantics from multi-view images

For each image, LightSplat collects SAM masks and their corresponding CLIP features, creating a clean 2D semantic inventory before any 3D reasoning begins.

2
Inject compact 2-byte indices into 3D

Instead of attaching full language vectors to every Gaussian, the method assigns only the most influential mask index to each relevant Gaussian.

3
Filter noise using geometric support

Masks with weak 3D support are removed, which suppresses view-dependent artifacts and stabilizes the semantics that survive into 3D.

4
Cluster semantically related masks in one step

Geometric overlap and semantic similarity are combined to build object-level clusters, giving the model a compact and interpretable 3D representation.

Fast Inference by Design

Once cluster-level features are formed, inference becomes much simpler. The text query is compared with a compact set of cluster features instead of every Gaussian or every rendered pixel.

This is why LightSplat is not just faster at distillation, but also faster at query time. The expensive part of reasoning is moved away from Gaussian-level comparisons and into a much smaller object-level representation, reducing similarity computations to below 0.1% of naive Gaussian-level inference depending on scene complexity.

Figure 3. Text is matched against cluster-feature mappings, not dense Gaussian features, which keeps the inference path short and scalable.

Qualitative Results

The qualitative results complete the story told by the benchmark numbers. Because semantics are organized at the object-cluster level rather than blurred across many Gaussians, LightSplat produces cleaner boundaries, more stable object identity, and broader scene coverage.

Ablation Study

The ablation results reinforce the design logic of the method. The speed stays almost unchanged across variants, while the quality drops sharply when 3D-aware mask filtering, semantic-aware clustering, or geometry-aware clustering is removed.
