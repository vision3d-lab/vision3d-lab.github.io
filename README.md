# vision3d-lab.github.io


## Project Page Setup Guide (Example from Junsu)

> This is only an example for reference.

You can use the following template to create your project page.  
The overall workflow is: **clone → customize → rename → upload**.

### 1. Template Repository
Use the following template as a starting point:

- https://github.com/eliahuhorwitz/Academic-project-page-template

### 2. Setup Steps
1. Clone the repository to your local machine.  
2. Modify the content (text, images, videos, etc.) to match your paper.  
3. Rename the top-level folder to your paper acronym (e.g., `soundact`).  
4. Upload the folder as your project page.
